\# DesktopAnimation



タスクバー付近にキャラクターを常時表示し、  

アイドルアニメーション・ゴーゴーアニメーションを再生する  

デスクトップマスコットアプリです。



本アプリは \*\*素材をユーザー自身が用意する方式\*\* のため、  

著作権的に安全に利用できます。



\## ✨ 主な機能



\- アイドルアニメーション（idle）を常時ループ再生

\- ゴーゴータイム（gogo）切り替え機能

\- ゴーゴー開始演出（gogoStart）対応

\- キャラクター切り替え（config.json）

\- 1P / 2P の向き切り替え

\- \*\*位置調整スライダー（リアルタイム反映）\*\*

\- PNG 連番アニメーションのプリロード方式で高速・安定動作



\## 📁 フォルダ構造

```
&nbsp;        │    └─ ...

&nbsp;        ├─ gogo/

&nbsp;        │    └─ ...

&nbsp;             ├─ 0.png

&nbsp;             └─ ...

```



\### 🔹 画像ファイルについて



\- 0.png / 1.png / 2.png … の \*\*連番 PNG\*\* を配置してください  

\- フォルダが空でも動作します（そのアニメはスキップされます）



\## ⚙ config.json の書き方



```json

{

&nbsp; "characters": \[

&nbsp;   { "id": 0, "name": "ドンちゃん" },

&nbsp;   { "id": 1, "name": "カスタムキャラA" }

&nbsp; ]

}

```



`id` と同じ番号のフォルダを `Flames/` に作成してください。



\## 🖥 使い方



\### 1. アプリを起動  

デスクトップ右下付近にキャラが表示されます。



\### 2. 設定画面を開く  

\*\*Ctrl + Shift + D\*\* を押すと設定画面が開きます。



\### 3. 設定できる項目



\- GoGo（サビ演出）ON/OFF  

\- 1P / 2P の向き  

\- キャラクター選択  

\- \*\*位置調整（X/Y スライダー） → リアルタイム反映\*\*



\## 📝 位置調整について



本アプリは環境（解像度・DPI・タスクバー位置）により  

キャラの表示位置が変わる場合があります。



設定画面の \*\*X/Y スライダーを動かすと即座に位置が変わる\*\* ため、  

ユーザー自身で最適な位置に調整できます。



\## 🚀 パフォーマンスについて



本アプリは \*\*全フレームを起動時にプリロード\*\* する方式を採用しています。



\- 初回キャラ読み込み時のみ数秒かかる場合があります  

\- 読み込み後は初回よりは早くでアニメーションが動作します  

\- フレーム数が多いほど初回読み込みが長くなります（仕様）
